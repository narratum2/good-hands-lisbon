**How to Fix**:
1. Go to https://vercel.com/narratums-projects/good-hands-lisbon
2. Settings → Environment Variables
3. Add:
   ```
   NOTION_API_KEY=secret_xxxxx
   NOTION_BOOKING_DATABASE_ID=xxxxx
   NOTION_CHAT_DATABASE_ID=xxxxx
   ```
4. Click "Redeploy"
