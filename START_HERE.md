## 🎯 COMMON TASKS

### **Deploy to Production:**
```bash
1. Check VERCEL_ENV_VARIABLES_CHECKLIST.md
2. Run: npm run build
3. Push to main branch
4. Vercel auto-deploys
```

### **Add New Content:**
```bash
1. Update Notion databases (see NOTION_COMPLETE_CMS_GUIDE.md)
2. Content syncs automatically
3. Or create .md files in content/blog/
```

### **Setup Automation:**
```bash
1. Choose: Make.com or n8n
2. Follow: make-scenario-templates.md or N8N_WORKFLOWS.md
3. Connect to Notion + Stripe
```

### **Troubleshooting:**
```bash
1. Check: troubleshooting-common-issues.md
2. Verify: Environment variables
3. Test: Notion API connection
```
